5.7.  3.7Math Font Size

In math mode, TE X selects the font size according to the context. Superscripts, for example, get typeset in a smaller font. If you want to typeset part of an equation in roman, don’t use the \textrm command, because the font size switching mechanism will not work, as \textrm temporarily escapes to text mode. Use \mathrm instead to keep the size switching mechanism active. But pay attention, \mathrm will only work well on short items. Spaces are still not active and accented characters do not work. [33]

 \begin{equation}  2^{\textrm{nd}} \quad  2^{\mathrm{nd}}  \end{equation}

(3.10)

Sometimes you still need to tell LaTeX the correct font size. In math mode, this is set with the following four commands:

\displaystyle ( ), \textstyle ( ), \scriptstyle ( )and \scriptscriptstyle ( ).

Changing styles also affects the way limits are displayed.

 \begin{displaymath}   \frac{\displaystyle     \sum_{i=1}^n(x_i-\overline x)     (y_i-\overline y)}    {\displaystyle\biggl[   \sum_{i=1}^n(x_i-\overline x)^2  \sum_{i=1}^n(y_i-\overline y)^2  \biggr]^{1/2}}  \end{displaymath}

This is an examples with larger brackets than \left[  \right] provides. The \biggl and \biggr commands are used for left and right brackets respectively.
