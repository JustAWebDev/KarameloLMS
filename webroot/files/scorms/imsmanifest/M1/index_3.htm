Key Words and Concept:

  1. Gesture: moving the hand or head to show what you want, feel, think, etc.

  2. Numeral: number; figure

  3. Refer: to describe something or be about something

  4. Symbol: mark, sign, or picture that shows something

 
 

 Instructional Objectives:

1. Learning some new words such as refer, symbol, gesture, numeral, etc.

2. Learning about different types of numbers

Numbers

Read the following selection to get a general understanding of it, then answer the questions that follow the passage.

A number is an idea that is used to refer to amounts of things. People use number words, number gestures, and number symbols. Number words are said loud. Number gestures are made with some parts of the body, usually the hands. Number symbols are marked or written down. A number symbol is called a numeral. When we read a numeral out loud, we say the number word for that numeral. When we want to write a number, we can write the numeral or the word. For example, we say the word five and write the numeral 5 or the word five. The number is the idea we think or when we see the numeral or when we see or hear the word.
